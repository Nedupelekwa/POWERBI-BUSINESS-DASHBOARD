# Business Intelligence & Sales Performance Dashboard

## Project Overview
This repository hosts a Power BI-driven analytics platform designed to monitor and evaluate corporate performance. The project focuses on visualizing sales dynamics, regional distribution, and key financial indicators to support data-driven decision-making.

## Features
- **Dynamic KPI Scorecard:** Real-time tracking of Revenue, Gross Profit, and Sales Targets.
- **Geospatial Analytics:** Interactive maps for regional sales distribution and market density.
- **Trend Forecasting:** Time-series analysis to identify seasonal peaks and growth patterns.
- **Granular Filtering:** Multi-level slicers for specific timeframes, regions, and product lines.

## Technical Architecture
- **Tooling:** Microsoft Power BI Desktop.
- **Data Modeling:** Implementation of a Star Schema to ensure high-performance query execution and data integrity.
- **DAX Logic:** Developed custom Measures for Year-to-Date (YTD) growth and moving averages.
- **UI/UX Design:** Optimized layout for executive presentations with a high data-to-ink ratio.

## Installation & Usage
1. Download the `Business Dashboard.pbix` file.
2. Open the file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
3. Interact with the visualizations by selecting specific filters on the left-hand navigation pane.

## Key Outcomes
- Streamlined the reporting process from manual spreadsheets to automated visual insights.
- Provided visibility into underperforming regions, allowing for targeted marketing interventions.

Project Summary: Executive Performance Overview

This project delivers a comprehensive Business Intelligence solution that tracks organizational health through a centralized Power BI dashboard. By integrating sales, regional, and product data, the dashboard transforms complex datasets into actionable executive insights.

Key Metrics & Findings:

Revenue Growth: Identified core revenue drivers across multiple product categories, highlighting a consistent upward trend in total sales volume.

Regional Performance: Top-performing territories were pinpointed using geospatial mapping, revealing significant market penetration in primary urban hubs.

Operational Efficiency: Analysis of order fulfillment cycles showed a reduction in processing lag, correlating with improved customer satisfaction scores.

Product Insights: A small subset of "hero products" accounts for a disproportionate share of total profit, suggesting an opportunity for inventory optimization.
